  • replace(oldValue, newValue): StringBuilder

  • 将此实例中出现的所有指定字符串的替换为其他指定字符串。

    参数

      要替换的字符串。

    • newValue: String

      替换 oldValue 的字符串。

    返回 StringBuilder

    对此实例的引用，其中 oldValue 的所有实例被 newValue 替换。

  • replace(oldValue, newValue, startIndex, count): StringBuilder

  • 将此实例的子字符串中出现的所有指定字符串替换为其他指定字符串。

    参数

    • oldValue: String

      要替换的字符串。

    • newValue: String

      替换 oldValue 的字符串。

    • startIndex: Number

      此实例中子字符串开始的位置。

    • count: Number

      子字符串的长度。

    返回 StringBuilder

    对此实例的引用，其中从 oldValue 到 newValue + startIndex - 1 的范围内 startIndex 的所有实例被 count 替换。
