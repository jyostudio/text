    • convert(
          srcEncoding: Encoding,
          dstEncoding: Encoding,
          bytes: Uint8Array,
          index: number,
          count: number,
      ): Uint8Array

      将字节数组内某个范围的字节从一种编码转换为另一种编码。

      参数

      • srcEncoding: Encoding

        源数组 bytes 的编码。

      • dstEncoding: Encoding

        输出数组的编码。

      • bytes: Uint8Array

        要转换的字节数组。

      • index: number

        要转换的 bytes 中第一个元素的索引。

      • count: number

        要转换的字节数。

      返回 Uint8Array

      Uint8类型数组，其中包含将 bytes 中某个范围的字节从 srcEncoding 转换为 dstEncoding 的结果。

      继承自 Encoding.convert

    静态getEncoding

    • getEncoding(codePage: number): Encoding

      返回与指定代码页标识符关联的编码。

      参数

      • codePage: number

        首选编码的代码页标识符。 可能值均在 Encoding 类主题中出现的表的“代码页”一列中列出。 - 或 - 0（零），使用默认编码。

      返回 Encoding

      与指定代码页关联的编码。

      抛出

      未找到编码。

    • getEncoding(name: string): Encoding

      返回与指定代码页名称关联的编码。

      返回 Encoding

      与指定代码页关联的编码。

      抛出

      name 不是有效的代码页名称。 - 或 - name 指定的代码页不受基础平台支持。

      继承自 Encoding.getEncoding
