# @jyostudio/text

提供了文本处理能力。

## 引用

浏览器

```HTML
<script type="importmap">
  {
    "imports": {
      "@jyostudio/text": "https://unpkg.com/@jyostudio/text"
    }
  }
</script>
```

Node.js

```bash
npm install @jyostudio/text
```

根据环境引用后，用法完全一致，不需要在使用时区分引用地址和方式。
